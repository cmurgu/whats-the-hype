A Header block contains helpful metadata; the Content block contains… content. The information in these blocks is what we want to extract and analyze.
Using specific software, we can process WARCs to extract what we want. For example, we can request a “full text derivative,” which would provide us with all of the text content (like the chunk below). We could then take this text and analyze it in Python or R.
