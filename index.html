The rectangle to the right represents a WARC file, or container. A WARC file is the output from a web crawler. 
A WARC is comprised of one or more WARC records.

WARC record
Header
The WARC record includes a ‘header’ and a ‘content' block. The header document contains mandatory metadata, such as the date, type, and length of the record and support the convenient retrieval of each harvested resource.
Content
The ‘content’ block includes the data that you may find on a specifc web page, in any format: a JPEG image, or text.
